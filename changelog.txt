Hades Accessibility Mods Installer Changelog

*Version 1.8: Released 03/11/2026
Language files are now automatically installed from the release zip to the game's x64/languages/ folder.
License files are now installed to the game's x64/ folder and the View License buttons open the local files instead of GitHub.

*Version 1.7: Released 03/09/2026
Added "View License" and "View Original License" buttons that open the license files on GitHub in your browser.

*Version 1.6: Released 03/03/2026
The installer now detects and removes old legacy mod files during installation. This includes the Content/Mods/ folder, modimporter.py, and mod.log. These files are no longer needed since all mods are now embedded in the DLL, and leaving them behind can cause conflicts such as game freezes from dual ModUtil versions.

*Version 1.5: Released 03/03/2026
Installer self-update now replaces the exe in-place instead of creating a new file with the version number in the name.

*Version 1.4: Released 03/03/2026
Mod releases are now downloaded as a single zip file instead of individual files. This is faster and ensures all files come from the same release.
The installer automatically extracts the DLL files to the game's x64 folder and optionally saves the readme and changelog.

*Version 1.3: Released 03/03/2026
Mod files are now downloaded from GitHub releases instead of the main branch. This ensures you always get a tested, published version.
Added mod update checking. When you open the installer, if a newer mod version is available it will show you the changelog and offer to install the update.
Fixed installer self-update checking the wrong GitHub repository.

*Version 1.2: Released 03/03/2026
Replaced "Copy Debug Log" button with "Open Debug Log Folder" button. Debug logs are now stored in a logs subfolder with timestamped filenames, so opening the folder is more useful than copying a single file.

*Version 1.1: Released 03/02/2026
Added automatic update checking. The installer now checks GitHub for newer installer versions on startup and offers to download them.
Changed the readme and changelog save location from the desktop to the folder the installer is run from.
Fixed NVDA speech not working (incorrect function names nvdaControllerClient64_speakText changed to nvdaController_speakText).

*Version 1.0: Released 03/02/2026
Initial release.
Downloads and installs the Hades accessibility mod DLL files from GitHub.
Automatically detects the Hades game directory via Steam registry entries.
Optional readme and changelog saving.
Copy Debug Log button for copying the accessibility log to clipboard.
NVDA screen reader support for installer status messages.
